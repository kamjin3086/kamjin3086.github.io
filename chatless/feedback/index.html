意见反馈

您的反馈是我们改进 Chatless 的重要动力。无论是遇到问题、功能建议，还是使用心得，都欢迎通过以下渠道告诉我们。

🐛 报告问题 在 GitHub Issues 提交 Bug 报告，附上复现步骤、系统信息与日志截图 💡 功能建议 在 GitHub Discussions 分享您的想法，参与功能讨论与投票

提交反馈前的建议

  • ✓先查阅常见问题，问题可能已有解决方案
  • ✓搜索现有 Issues，避免重复提交
  • ✓提供详细信息：操作系统、应用版本、复现步骤
  • ✓使用清晰的标题，便于问题分类与检索
