简介

Chatless 是一款本地优先的 AI 聊天桌面应用，基于 Tauri 2.0 与 Next.js 构建。无论您选择云端 AI 服务商（如 OpenAI、Anthropic、DeepSeek），还是在本地运行模型（通过 Ollama 等），Chatless 都能为您提供流畅、安全、可控的对话体验。

快速开始 GitHub 下载安装

多提供商支持

一键切换云端 AI 服务或本地模型，配置灵活、管理集中。

启用检索 管理知识库

MCP 服务 Prompt 管理

三步开始使用

  1. 下载并安装适合您系统的安装包（Windows / macOS / Linux）
  2. 打开应用，在设置中添加 AI 提供商密钥或配置本地模型
  3. 选择模型，开始智能对话（可上传文档或启用知识库）
