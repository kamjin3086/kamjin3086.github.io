Chatless 简介

Chatless 是一个基于 Tauri 2.0 与 Next.js 构建的现代 AI 聊天桌面应用。支持多种 AI 提供商，亦可连接 Ollama 等本地模型；提供文档与图片解析、RAG 本地知识库、Prompt 管理与 MCP 工具集成，跨平台、轻量且启动迅速，所有数据本地存储以保障隐私。
