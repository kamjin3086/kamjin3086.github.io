使用说明

使用说明

首次配置

在开始使用 Chatless 之前，请完成以下基础配置：

  1. 配置 AI 提供商：在设置中添加您的 API 密钥（如 OpenAI、Anthropic、DeepSeek、Gemini 等）
  2. 配置本地模型（可选）：如需离线使用，可安装 Ollama 并下载模型，然后在设置中配置其地址
  3. 创建知识库（可选）：上传 PDF、Word 或 Markdown 文档，构建本地知识库以启用 RAG 检索增强对话

开始对话

完成配置后，您可以：

  1. 选择模型：在聊天界面顶部下拉菜单中选择您想使用的 AI 模型
  2. 输入问题：在输入框中输入您的问题或指令，按 Enter 或点击发送
  3. 附加文件：点击附件按钮上传图片或文档，进行基于文件的问答与分析
  4. 使用 Prompt：输入 / 快速插入预设提示词模板
  5. 调用工具：输入 @ 调用已配置的 MCP 服务

管理历史记录

您可以在历史记录页面：

  • 查看所有对话记录，按时间或关键词搜索
  • 导出特定对话为 Markdown 或 JSON 格式
  • 删除不需要的对话记录
