对话与工具

Chatless 支持多种对话功能，包括常规对话、视觉模型对话、文生图对话、文档解析对话等。也支持使用提示词对话、调用MCP服务、数学渲染等功能。
在开始对话前先设置AI提供商以及拉取、添加模型。

常规对话

配置模型后右上角选择模型开始对话。

支持常规模型，对文字进行理解与回答，适合文字题等场景。

视觉模型对话

支持ollama等提供商的视觉模型，支持分析图片等功能，需要在ollama下载支持视觉分析的多模态模型，例如qwenvl系列、gemma3系列、llava系列等。

支持 Vision 模型，对图片进行理解与回答，适合图表、界面、视觉题等场景。

文生图对话

支持gemini等提供商的文生图模型，使用提示词直接让其生成图片。

支持文生图模型，对图片进行生成与回答，适合图表、界面、视觉题等场景。

文档解析对话

点击输入框中的附加文档按钮，选择一个支持的文档格式（pdf、word、markdown等）完成文档解析，之后发送给大模型进行对话。注意文档的字数和大小都不宜太大，该功能受到解析性能和模型性能制约。

支持 PDF / Word / Markdown 文档解析，对文档内容进行问答。

数学渲染与格式

对数学公式与代码块进行良好渲染，便于阅读与复核。

调用 MCP 对话

首先在MCP 服务器设置中导入mcp服务，然后可以在对话中使用@调用mcp服务（建议主动@mcp以提升成功调用的几率，否则可能部分模型的mcp服务调用成功率不高）。

通过 MCP（Model Context Protocol）统一接入外部工具链，在对话中便捷调用。

使用提示词对话

在对话中使用/打开提示词面板，选择提示词开始对话。

使用提示词，在对话中便捷调用。
