设置

AI 提供商设置

在设置中添加各家提供商的 API 密钥，主动刷新模型列表，点击齿轮按钮可以添加或取消提供商的显示，添加自定义提供商等。

AI 提供商面板用于管理云端密钥与本地模型地址，配置后可在对话中切换与使用。

管理提供商

勾选需要的提供商，如果没有找到提供商可以自定义提供商。

AI 提供商管理面板，支持添加或取消提供商的显示，添加自定义提供商等。

知识库设置

嵌入模型管理-Ollama服务需要首先在提供商中配置后才可以使用，本地模型则是直接下载即可使用，激活后可以点击闪电按钮进行嵌入服务连接情况测试。之后添加知识库以开始使用

用于配置向量化、切分与解析等参数，提升检索与引用的质量。

MCP 服务器设置

用于管理 MCP 服务的工具源与权限，集中接入第三方工具链。
