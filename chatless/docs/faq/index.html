FAQ

常见问题（FAQ）

1.Chatless 支持哪些平台？

Chatless 提供 Windows、macOS 和 Linux 三大平台的原生桌面应用。各平台均支持最新的主流版本。

2.Windows 启动后立即退出或闪退，怎么办？

这通常是因为系统缺少 Microsoft Visual C++ 运行库。请按以下步骤解决：

  1. 下载并安装 Microsoft Visual C++ 2015-2022 Redistributable (x64)
  2. 安装完成后重启 Chatless 应用
  3. 如问题仍未解决，请在应用内"设置 → 反馈"提交日志与详细描述，或访问 GitHub Issues 寻求帮助

3.Chatless 会上传我的数据吗？

不会。Chatless 采用本地优先原则，所有对话记录、文档、知识库和配置均存储在您的本地设备，不会上传至任何服务器。您可以随时查看开源代码进行验证。

4.是否支持本地模型（离线使用）？

支持。您可以通过 Ollama 或 LM Studio 在本地运行 AI 模型，并在 Chatless 设置中配置其地址后使用。详见本地模型配置。

5.如何下载和安装 Chatless？

访问 GitHub Releases 页面下载最新版本：立即下载。选择适合您系统的安装包（.exe / .msi / .dmg / .AppImage 等）并按提示安装。

6.遇到问题或有建议，如何反馈？

我们提供多种反馈渠道：

  • GitHub Issues：提交 Bug 报告或功能请求
  • GitHub Discussions：参与社区讨论与交流
  • 应用内反馈：在"设置 → 反馈"页面提交（可附带日志文件，便于问题排查）

7.Chatless 是免费的吗？需要订阅吗？

Chatless 完全免费且开源（MIT License），无需订阅。您只需为您选择的 AI 提供商服务（如 OpenAI、Anthropic）付费，或使用完全免费的本地模型。
