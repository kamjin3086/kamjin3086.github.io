Prompt 管理

对话侧快捷面板

提示词支持设置为系统提示词，只需要补全提示词变量后，使用Alt+Enter。

在对话侧可以用 `/` 打开快捷面板，插入常用提示词，加速构思与一致性。

提示词管理

管理提示词，包括添加、删除、编辑提示词，以及导入提示词等。
