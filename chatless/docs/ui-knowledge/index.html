知识库（RAG）

Chatless 支持方便的管理知识库，支持按关键字快速检索历史对话，查看详情并导出为文件，便于归档与分享。
首先需要完成知识库设置，导入文档，之后就可以在对话中使用知识库了。

知识库管理

管理知识库，包括添加、删除、编辑知识库，以及导入文档等。

知识库详情

展示单篇文档的解析与片段信息，便于复核数据来源。

知识库对话

对话时按需检索本地向量索引，引用相关片段，提高回答准确性与可追溯性。
