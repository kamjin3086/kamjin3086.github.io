从源码构建

本文档说明如何在本地克隆与构建 Chatless 桌面应用。

环境要求

  • Node.js 18+
  • pnpm（推荐）或 npm
  • Rust 工具链（Tauri 2.0 需要）
  • 平台依赖：Windows（Visual Studio Build Tools / MSVC）、macOS（Xcode Command Line Tools）、Linux（libgtk-3-dev、webkit2gtk 等，见 Tauri 官方）

克隆与安装

git clone https://github.com/kamjin3086/chatless.git
cd chatless
pnpm install

开发调试

pnpm tauri dev

生产构建

pnpm tauri build

产物位于 src-tauri/target/release（或平台对应目录）。具体脚本名称以仓库 package.json 为准，如使用 npm run tauri:dev/tauri:build 亦可。
