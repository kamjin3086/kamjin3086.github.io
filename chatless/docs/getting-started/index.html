快速开始

下载安装

点击进入下载页面，选择适合您系统的安装包：

  • Windows: Releases 最新版（.exe / .msi）
  • macOS: Releases 最新版（aarch64 / x64）
  • Linux: Releases 最新版（.rpm / .deb / .AppImage）
详细安装说明：参见仓库中的 安装指南。

Windows 闪退（缺少 VC++ 运行库）

若启动即退出/闪退，通常是缺少 Microsoft Visual C++ 运行库：

  1. 下载 Microsoft Visual C++ 2015–2022 Redistributable (x64)： vc_redist.x64.exe
  2. 安装后重启应用（必要时重启系统）。
  3. 如仍异常，请在“设置 → 反馈”提交日志与描述。

参考： #23 (comment) 评论感谢 @ukhack 提供解决方案。

配置 AI 提供商

在应用设置 → AI 模型中添加您选择的提供商 API 密钥：

  1. 打开设置界面，进入"AI 模型"页签
  2. 点击"添加提供商"，选择 OpenAI、Anthropic、DeepSeek 或 Gemini 等
  3. 填入对应的 API Key 并保存，应用将自动刷新可用模型列表

配置本地模型（可选）

如需使用本地模型，推荐通过 Ollama 运行。以下是快速配置示例：

# 1. 安装 Ollama（macOS / Linux）
curl -fsSL https://ollama.ai/install.sh | sh

# 2. 拉取常用模型（根据需要选择）
ollama pull llama2:7b      # Meta Llama 2，7B 参数
ollama pull mistral:7b     # Mistral AI，性能优秀
ollama pull qwen:7b        # 通义千问，中文支持好

# 3. 在 Chatless 设置中填入 Ollama 地址（默认 http://localhost:11434）

详细配置说明请参考配置与提供商文档。

从源码构建（开发者）

如需参与开发或自定义构建，可按以下步骤操作：

# 克隆仓库
git clone https://github.com/kamjin3086/chatless.git
cd chatless

# 安装依赖（使用 pnpm）
pnpm install

# 启动开发模式
pnpm tauri dev

更多开发信息请查看从源码构建和贡献指南。
