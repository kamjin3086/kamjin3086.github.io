常见问题

Chatless 是否完全免费？

Chatless 本身是完全免费开源的，但使用第三方 AI 服务（如 OpenAI）需要您自己的 API Key 和相应费用。您也可以使用完全免费的本地模型。

我的数据会被上传吗？

不会。Chatless 采用本地优先设计，所有对话记录、配置文件都存储在您的本地设备上。使用第三方 API 时，对话内容会发送到相应服务商，但 Chatless 本身不会收集或上传任何数据。

支持哪些 AI 模型？

支持 OpenAI、Anthropic、DeepSeek、Gemini、Ollama 等多种服务商，同时支持本地运行的开源模型。详见配置文档。
