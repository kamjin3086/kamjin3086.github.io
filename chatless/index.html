开源、轻量级、现代化。本地 AI 对话客户端，支持多提供商与本地模型。

开源 MIT 轻量级 现代化 隐私友好
GitHub 仓库 下载页面
Windows EXE / MSI 安装包 macOS DMG / ZIP 应用 Linux AppImage / deb / rpm

  • 跨平台：Windows / macOS / Linux
  • 隐私友好：数据本地存储；开源 MIT 许可
