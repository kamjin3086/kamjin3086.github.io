Chatless 隐私政策

最近更新：2025 年 1 月

1. 信息收集

Chatless 遵循本地优先原则，默认情况下：

  • 所有对话记录、知识库数据及配置文件均存储在用户本地文件系统，不会上传至任何服务器
  • 本应用不会主动收集、分析或传输任何可识别个人身份的信息（PII）

2. 第三方 API 调用

当您自行配置第三方 AI 提供商 API Key 时，请注意：

  • 请求内容及上下文将直接发送至相关 API 服务商的服务器
  • 这些数据的处理将受相应服务商的隐私政策约束，而非 Chatless
  • 请仔细阅读并遵守各服务商政策，必要时对敏感数据进行脱敏处理

3. Cookies 与本地存储

Chatless 不使用浏览器 Cookie。为实现持久化设置与离线功能，会在本地存储（如 SQLite、文件系统）写入必要的配置与缓存文件，这些数据仅在本地可见。

4. 数据安全

我们采用加密存储（如 SQLCipher 或系统加密 API）保护重要数据；同时建议您自行加密硬盘或使用系统级备份方案。

5. 未成年人保护

若您未满 14 周岁，请在监护人同意并指导下使用本应用。

6. 用户权利

您可随时：

  • 在应用设置中导出或删除本地数据
  • 撤销第三方 API Key
  • 卸载应用并删除相关数据文件

7. 政策更新

当我们对本隐私政策进行重大变更时，将在应用内及本页面同步通知。更新后继续使用即视为接受修改。

8. 联系我们

关于隐私保护事宜，请通过 意见反馈 页面与我们联系。
