Crawell

全能网页素材采集，一键导出图片与 Markdown 内容

Chrome 商店 Edge 插件 Firefox 附加组件

核心功能

  • 一键抓取网页图片并自动分类
  • 将正文结构化保存为 Markdown
  • 批量下载 / ZIP 打包导出
  • 100% 本地处理，无登录、无上传

权限与隐私

Crawell 需要读取当前网页以提取素材，所有数据都仅在本地浏览器内处理，绝不会上传到任何服务器。

快速开始

  1. 安装插件 → 点击工具栏图标打开侧边栏
  2. 选择「图片」或「内容」模块开始采集
  3. 筛选后下载、或直接复制 Markdown
